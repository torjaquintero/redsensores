# Ejercicio No. 1

Este programa implementa una **estación básica de monitoreo ambiental** basada en un Arduino UNO R4 WiFi que integra múltiples sensores analógicos y digitales para observar las condiciones de una planta o entorno agrícola. El sistema adquiere datos de **calidad del aire (MQ135), presencia de gases combustibles (MQ9) y humedad del suelo** mediante entradas analógicas, mientras que la **temperatura y humedad ambiental** se obtienen con el sensor SI7021 y el **nivel de iluminación** con el sensor BH1750 a través del bus I2C. Durante la fase de inicialización se verifica la correcta conexión de los sensores digitales y, una vez validado el sistema, el programa realiza una **lectura periódica cada 2 segundos**, enviando todos los valores al **monitor serial** para permitir la supervisión en tiempo real del estado ambiental del sistema de sensores.

# Ejercicio No. 2

Este programa demuestra cómo integrar un **reloj en tiempo real** basado en el módulo DS3231 RTC Module con un sistema embebido utilizando comunicación I2C. El código inicializa el dispositivo, verifica su correcta conexión y comprueba si el reloj perdió alimentación; en ese caso, ajusta automáticamente la fecha y hora utilizando la información de compilación del programa. Una vez configurado, el sistema obtiene continuamente la fecha y hora actual y las muestra en el monitor serial con un formato legible que incluye **día, mes, año y hora en formato HH:MM:SS**, actualizando la información cada segundo. Este ejercicio introduce los fundamentos para incorporar **referencias temporales precisas** en aplicaciones de registro de datos, monitoreo ambiental y sistemas IoT basados en plataformas como Arduino UNO R4 WiFi.

# Ejercicio No. 3

Este ejercicio implementa una **estación de monitoreo ambiental con registro temporal de datos** utilizando un Arduino UNO R4 WiFi y un módulo de reloj en tiempo real DS3231 RTC Module. El sistema adquiere información de varios sensores ambientales: **MQ135** para calidad del aire, **MQ9** para detección de gases combustibles, un **sensor de humedad del suelo**, además del sensor digital SI7021 para temperatura y humedad relativa y el sensor de iluminación BH1750. Cada lectura se asocia con la **fecha y hora exacta proporcionada por el RTC**, generando un registro estructurado que se muestra en el monitor serial. Este ejercicio introduce el concepto de **timestamp en sistemas embebidos**, una técnica fundamental para construir **data loggers, redes de sensores y sistemas de monitoreo ambiental**.
